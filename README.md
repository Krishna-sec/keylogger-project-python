# 🔐 Keylogger Project (Python)

## 📌 Description
This project is a simple keylogger built using Python. It captures keyboard inputs such as key press, hold, and release events.

The data is stored in:
- logs.json (structured format)
- logs.txt (continuous keystrokes)

A basic GUI is created using Tkinter to start the keylogger.

---

## ⚙️ Technologies Used
- Python
- Tkinter (GUI)
- Pynput (Keyboard Listener)
- JSON
- File Handling

---

## 🚀 Features
- Captures keystrokes in real-time
- Logs key press, hold, and release events
- Stores data in JSON and TXT files
- Simple GUI interface
